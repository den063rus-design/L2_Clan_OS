# L2 Clan OS

Публичная витрина Discord-бота для Lineage 2 кланов.

На странице показано:

- что такое L2 Clan OS;
- зачем он нужен кланам;
- полный мануал по тому, как бот работает;
- какие задачи закрывает бот;
- как выглядят кнопки, формы, рейды, DKP, лут и права;
- как подключить Discord-сервер и почему чужие серверы не получают доступ автоматически.

## GitHub Pages

Для публикации:

1. Открыть `Settings -> Pages`.
2. В `Build and deployment` выбрать `Deploy from a branch`.
3. Выбрать ветку `main`.
4. Выбрать папку `/root`.
5. Нажать `Save`.

После публикации страница будет доступна по адресу:

```text
https://den063rus-design.github.io/L2_Clan_OS/
```

Отдельная страница с полным мануалом:

```text
https://den063rus-design.github.io/L2_Clan_OS/manual.html
```
